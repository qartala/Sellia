# Plan de Modelos

1. **Gemini 3.1 Pro**: Emplear para comprensión profunda de requisitos de negocio, generación de estrategias publicitarias y optimización de flows complejos.
2. **gpt-oss-120b / Llama 3**: Utilizar para generación de código, respuestas comunes, análisis sintáctico y tareas que consuman muchos tokens.
3. **Mistral**: Integración para diversificar capacidades lingüísticas y de generación.
4. **Selector de Modelos**: Capa de enrutamiento que permite elegir el modelo según coste, velocidad y calidad.
