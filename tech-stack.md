# Tech Stack

- **Frontend**: React 19, TypeScript, Vite, Tailwind CSS v4
- **Routing**: React Router
- **Icons**: Lucide React
- **Charts**: Recharts
- **Animations**: Motion (Framer Motion)
- **Backend**: Node.js / Express (MVP API)
- **AI Integration**: `@google/genai` for Gemini 3.1 Pro (simulated multi-model routing for others)
